Sony GO FOR IT Q2 参考情報 (矢藤 康祐)

1. プログラムコード

https://github.com/kohyatoh/kohyatoh_contests/blob/master/sony/go_for_it/q02/q02int.f (i)
https://github.com/kohyatoh/kohyatoh_contests/blob/master/sony/go_for_it/q02/q02float.f (ii, iii)

2. 出力

i) 入力=5
 120
ii) 入力=2.5
      3.3233510
ii) 入力=-1.9
      1.8273551

3. アルゴリズムの簡単な説明

i)は定義通りの計算、ii), iii)はLanczos approximationを用いた近似 (GNU Scientific Libraryの移植)。

4. 動作方法の説明

Ubuntu 11.04での動作例
$ sudo apt-get install fort77
$ fort77 q02int.f
$ ./a.out
<入力>
$ fort77 q02float.f
$ ./a.out
<入力>
