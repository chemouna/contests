4. 動作方法の説明

Ubuntu 11.04での動作例
$ sudo apt-get install ghc
$ ghc --make -O3 q03slow.hs
$ ghc --make -O3 q03fast.hs
$ ghc --make -O3 gensample.hs
$ echo "sxoxnxy" | ./q03slow sony
2, 1
$ ./gensample | ./q03fast sony > answer.txt
